# The Inspector

## Opening the Inspector

There are many ways to open the inspector and it also depends on the *browser* that you are using.

- In **Chrome**; to open the Inspector you can press `F12` or `Right-Click` / `Mouse Button 2` and hit *Inspect Element*

- In **Opera** ( *the one that I personally use* ); to open the Inspector, you can press `Ctrl + Shift + C` or use the Mouse ( *YUCK!* )

- In **Obsidian**, yes, you can configure Obsidian with HTML and CSS, to open the Inspector, you can use the Shortcut `Ctrl + Shift + I`
